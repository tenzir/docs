# ocsf::type_name

Returns the `type_name` for a given `type_uid`.

```tql
ocsf::type_name(id:int) -> string
```

## Description

[Section titled “Description”](#description)

### `id: int`

[Section titled “id: int”](#id-int)

The `type_uid` for which `type_name` should be returned.
