## Description

The <code>{props.operator}</code> function prints a record's values as a
{props.separator_text} separated string.

### `input: record`

The record you want to print.

### `list_separator = str (optional)`

The string separating the elements in list fields.

Defaults to <code>"{props.list_sep}"</code>.

### `null_value = str (optional)`

The string denoting an absent value.

Defaults to <code>"{props.null_value}"</code>.
