Accepts bytes from standard input.

```tql
load_stdin
```

## Description

Accepts bytes from standard input. This is mostly useful when using the
`tenzir` executable as part of a shell script.

## Examples

### Pipe text into `tenzir`

```sh
echo "Hello World" | tenzir
```

```tql
load_stdin
read_lines
```

```tql
{
  line: "Hello World",
}
```
