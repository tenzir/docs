The `http` operator is deprecated. Use the dedicated HTTP operators instead.

## Migration guide

Choose the replacement that matches how you used `http`:

- <Op>from_http</Op>: make outbound requests and parse the response into
  events.
- <Op>to_http</Op>: send events as a single outbound request to a webhook or API.
- <Op>accept_http</Op>: start a server that listen for inbound requests
  and process their bodies.
- <Op>serve_http</Op>: start a server that streams pipeline output to
  clients.
