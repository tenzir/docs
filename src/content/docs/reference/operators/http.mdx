The `http` operator is deprecated. Use the dedicated HTTP operators instead.

## Migration guide

Choose the replacement that matches how you used `http`:

- If you used `http` to make outbound requests and parse the response into
  events, use <Op>from_http</Op>.
- If you used `http` to send each event to a webhook or API, use
  <Op>to_http</Op>.
- If you used `http` to listen for inbound requests and process their bodies,
  use <Op>accept_http</Op>.
- If you used `http` to start a server that streams pipeline output to clients,
  use <Op>serve_http</Op>.
