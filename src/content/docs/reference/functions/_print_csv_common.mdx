## Description

<span> The <code>{props.operator}</code> function prints a record's values as a {props.separator_text} separated string.</span>

### `input: record`

The record you want to print.

### `list_separator = str (optional)`

The string separating different elements in a list within a single field.

<span>Defaults to <code>"{props.list_sep}"</code></span>.

### `null_value = str (optional)`

The string denoting an absent value.

<span>Defaults to <code>"{props.null_value}"</code></span>.
