Checks if a string follows title case.

```tql
is_title(x:string) -> bool
```

## Description

The `is_title` function returns `true` if `x` is in title case and `false`
otherwise.

## Examples

### Check if a string is in title case

```tql
from {x: "Hello World".is_title()}
```

```tql
{x: true}
```
