The Tenzir package includes two binaries:

- **`tenzir`**: Execute a single pipeline from the command line
- **`tenzir-node`**: Run a persistent node for continuous pipeline execution

These binaries run pipelines on your machine. To manage pipelines through a web
interface, [create an account](/guides/installation/create-account) and use the
[platform](https://app.tenzir.com). See the [architecture
overview](/explanations/architecture) to learn how they fit together.

:::tip[Supported Platforms]
Tenzir runs on **macOS** (Apple Silicon) and **Linux** (x86_64, ARM64).
:::

Choose the installation method that fits your use case.

## uvx

Run pipelines instantly without installation:

```sh
uvx tenzir 'from {x: 42}'
```

Use `uvx tenzir@latest` to ensure you always get the most recent release. If
startup time is a concern, omit `@latest` and manually refresh the cache when
you want updates.

## curl installer

Run the installer and follow the instructions to download the binaries, and
optionally start a node:

```sh
curl https://get.tenzir.app | sh
```

The installer script asks for confirmation before performing the installation.
It uses the package manager of your Linux distribution to install the Tenzir
package. This typically also creates a [systemd](https://systemd.io) unit and
starts the node automatically.

## Packages

<Tabs>
<TabItem value="debian" label="Debian" icon="linux">

<Steps>

1. Download the latest [Debian package][tenzir-debian-package-amd64] ([arm64 version][tenzir-debian-package-arm64]).

2. Install it via `dpkg`:
   ```sh
   dpkg -i tenzir_latest_arm64.deb
   ```

</Steps>

To uninstall, run `apt-get remove tenzir`. Use `purge` instead of `remove` to
also delete the state directory.

[tenzir-debian-package-amd64]: https://storage.googleapis.com/tenzir-dist-public/packages/main/debian/tenzir_latest_amd64.deb
[tenzir-debian-package-arm64]: https://storage.googleapis.com/tenzir-dist-public/packages/main/debian/tenzir_latest_arm64.deb

</TabItem>
<TabItem value="rpm" label="RPM-based" icon="linux">

<Steps>

1. Download the latest [RPM package][tenzir-rpm-package] ([aarch64 version][tenzir-rpm-package-aarch64]).

2. Install it via `rpm`:
   ```sh
   rpm -i tenzir-latest-x86_64-linux-static.rpm
   ```

</Steps>

[tenzir-rpm-package-amd64]: https://storage.googleapis.com/tenzir-dist-public/packages/main/rpm/tenzir-latest-x86_64-linux-static.rpm
[tenzir-rpm-package-aarch64]: https://storage.googleapis.com/tenzir-dist-public/packages/main/rpm/tenzir-latest-aarch64-linux-static.rpm

</TabItem>
<TabItem value="nix" label="Nix" icon="nix">

Run Tenzir ad-hoc without installing:

```sh
nix run github:tenzir/tenzir/latest
```

Or enter a shell with Tenzir available:

```sh
nix shell github:tenzir/tenzir/latest -c tenzir 'from {x: 42}'
```

Install a Tenzir Node by adding `github:tenzir/tenzir/latest` to your flake
inputs, or use your preferred method to include third-party modules on classic
NixOS.

:::note[Pre-built only!]
The default attributes of the flake contain some closed source plugins and will
fail to download locally. The flake is only for public consumption together with
the [Tenzir cachix cache](https://tenzir.cachix.org).
:::

</TabItem>
<TabItem value="docker" label="Docker" icon="seti:docker">

Run pipelines in a container without installing anything locally:

```sh
docker run --rm -it tenzir/tenzir 'from {x: 42}'
```

</TabItem>
<TabItem value="tarball" label="Tarball" icon="seti:folder">

<Steps>

1. Download the [static binary tarball][tenzir-tarball].

2. Unpack it into `/opt/tenzir`:
   ```sh
   tar xzf tenzir-latest-x86_64-linux-static.tar.gz -C /
   ```

</Steps>

[tenzir-tarball-amd64]: https://storage.googleapis.com/tenzir-dist-public/packages/main/tarball/tenzir-latest-x86_64-linux-static.tar.gz
[tenzir-tarball-aarch64]: https://storage.googleapis.com/tenzir-dist-public/packages/main/tarball/tenzir-latest-aarch64-linux-static.tar.gz

We also offer [prebuilt binaries for every commit to `main`][tenzir-tarball-main] ([aarch64 version][tenzir-tarball-main-aarch64]).

[tenzir-tarball-main-amd64]: https://storage.googleapis.com/tenzir-dist-public/packages/main/tarball/tenzir-main-x86_64-linux-static.tar.gz
[tenzir-tarball-main-aarch64]: https://storage.googleapis.com/tenzir-dist-public/packages/main/tarball/tenzir-main-aarch64-linux-static.tar.gz

</TabItem>
